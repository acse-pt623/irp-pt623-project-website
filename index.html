irp-pt623 Documentation¶

The irp-pt623-externel project is dedicated to predicting copper prices using advanced machine learning models such as BNN-CNN-LSTM and Multi-TCN-LSTM-Attention. This documentation will help you understand how to install, use, and extend the various modules within this project.
